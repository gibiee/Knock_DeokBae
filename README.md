# 덕배 죽이기

## 소개
제가 초등학교 6학년 때 만든 조잡한 플래시 게임을 안드로이드로 이식해봤습니다.

당시 선생님과 학부모들이 모두 참관하는 자리에서 시연했는데, 특유의 폭력성으로 인해 큰 논란이 되었죠.

※ 어플의 완성도를 보고 욕이 나오셨다면 지극히 정상입니다.
